# UCX - Universal Code Executor

UCX is a powerful tool designed for running and merging scripts seamlessly across Android and desktop environments. It simplifies the development process, allowing developers to execute code snippets and scripts on multiple platforms without compatibility issues.

## Key Features
- **Cross-Platform Compatibility:** Run scripts on both Android and desktop.
- **Seamless Merging:** Easily merge scripts and manage code changes.
- **User-Friendly Interface:** Designed to make execution and management straightforward for developers.

## Installation
To get started with UCX, follow these steps:
1. Download the latest release from the [GitHub Releases page](https://github.com/devlost1990-cmd/ucx/releases).
2. Follow the installation instructions provided in the documentation.

## Usage
Once installed, you can execute scripts using the command line or the graphical interface, depending on your preference. Refer to the usage guide in the documentation for detailed instructions.
